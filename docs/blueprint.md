# **App Name**: VoxaLo Connect

## Core Features:

- User Authentication: Secure user registration and login with email/password and/or social accounts.
- Contact List: A directory of existing users for starting new chats
- One-to-One Chat: Real-time text-based messaging between two users.
- Real-time Updates: Update all the information (new users/messages/etc.) in real time using websockets.
- Notifications: In-app notifications for new messages.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to convey trust and communication.
- Background color: Light gray (#ECEFF1) for a clean, modern look.
- Accent color: Soft green (#81C784) for subtle highlights and call-to-actions.
- Body and headline font: 'Inter' sans-serif for clear, readable text.
- Simple, outline-style icons for common actions like send, delete, and settings.
- Clean, minimalist layout with clear separation between chat bubbles.
- Subtle animations for message sending and receiving.
